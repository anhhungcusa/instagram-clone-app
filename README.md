# Instagram clone (FE)

Clone Instagram with MERN Stack

## Requirements

- Install Node

## Using

- react
- react-router-dom
- antd

## Feature

- Add post
- like post, comment
- reply to comment
- notification

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

``` cmd
npm i
```

## Run

``` cmd
npm start
```
